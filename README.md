# Raylib Tetris Remake
A tetris remake made in C using RayLib for my [academy](https://github.com/aiv01)

**Features:**
Multiple Scenes Management,
Savedata (HighScore),
SFX & Music,
Pause Menu,
Effect when a line is completed.
